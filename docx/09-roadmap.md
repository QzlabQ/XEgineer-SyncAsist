## Phase 4：模板与 H5 样式引擎（预计 6-8 周）

**目标**：从 Markdown → 类 H5 富文本升级。核心是 CSS 样式引擎 + 平台模板系统。

**背景**：微信公众号文章本质上是微信内置浏览器的 H5 页面，支持完整的 CSS 内联样式（`<style>` 标签会被过滤，必须全部内联）。当前我们只生成基础 HTML，视觉效果远达不到公众号大号的专业排版水平。Phase 4 要做的不是"Markdown 渲染成 HTML"，而是"在编辑器中设计 H5 页面，导出为微信公众号兼容的内联样式 HTML"。

### CSS 内联样式引擎

- [ ] AST → 内联样式 HTML 生成器（替代当前简单的 HTML 拼接）
- [ ] 设计 Token 系统：配色 / 字体 / 间距 / 圆角 / 阴影 → 自动生成对应 CSS
- [ ] 响应式布局支持（Flexbox / Grid → 内联 style）
- [ ] 渐变背景、阴影卡片、圆角容器
- [ ] CSS 动画引擎：淡入、滑入、缩放，导出为 `animation` + 内联 `@keyframes`
- [ ] 点击交互（`:active` 伪类样式，模拟按钮反馈）
- [ ] 全宽 / 通栏段落（突破微信正文 677px 宽度限制的视觉效果）
- [ ] 富文本排版：首字下沉、竖排文本、文字环绕

### 微信公众号 H5 模板

- [ ] 卡片布局模板（标题卡 / 引用卡 / 作者卡 / 推荐卡）
- [ ] 图文混排模板（左图右文 / 上文下图 / 交错排列）
- [ ] 列表样式模板（带图标的编号列表、时间轴、步骤卡片）
- [ ] 引用块样式（对话气泡、书摘卡片、高亮引用）
- [ ] 分割线样式（渐变线、图标分隔、留白分隔）
- [ ] 表格模板（斑马纹、首列标题、自适应窄屏）
- [ ] 代码块：语法高亮渲染为图片
- [ ] 封面图自动裁剪（2.35:1）+ 多封面（大封面 / 小封面）
- [ ] 正文图片自动转存微信 CDN（mmbiz.qpic.cn）

### 小红书图文模板

- [ ] 多图排版（4/6/9 图布局，支持留白和圆角）
- [ ] 正文 emoji 段落点缀自动生成
- [ ] 封面图自动裁剪（3:4）
- [ ] 话题标签智能排序

### 模板系统

- [ ] 模板 DSL（JSON 定义模板参数：布局 / 配色 / 字体）
- [ ] 预设模板库（10+ 套公众号模板，5+ 套小红书模板）
- [ ] 品牌自定义模板（Logo / 配色 / 字体组合）
- [ ] 编辑器中实时模板切换预览

**Phase 4 交付物**：

- CSS 内联样式引擎：编辑器设计 → 微信公众号兼容 HTML
- 微信公众号 H5 模板库：卡片、引用、列表、分割线等 10+ 组件
- 小红书多图排版
- 可扩展的模板 DSL + 预设模板市场
