## 2. 各区域详细设计

### 2.1 顶部导航栏

```
[≡]  XEgineer    文章列表    设置          [Matrix ▾]  [草稿已保存 ✓]  [发布 ▼]
```

- **发布按钮**：点击展开下拉菜单，显示已选平台列表和"立即发布"按钮
- **保存状态**：实时显示"正在保存..."/"草稿已保存 ✓"/"保存失败 ✗"
- **账号**：点击展开账号管理

### 2.2 左侧面板

**平台选择区**：
- 复选框列表，显示平台图标 + 名称 + 登录状态
- 未登录的平台显示橙色警告图标，点击跳转登录
- 支持全选/全不选

**平台配置区**：
- 仅展示已勾选平台的配置项
- 折叠/展开设计，默认展开第一个平台
- 配置项根据 `metaSchema` 动态生成

### 2.3 编辑区

**标题输入框**：
- 大字体（32px），无边框，placeholder "文章标题"
- 按 Enter 或 Tab 跳转到正文编辑区

**Tiptap 编辑区**：
- 工具栏固定在编辑区顶部（悬浮工具栏 + 固定工具栏两种模式可切换）
- 支持 `/` 命令快速插入块（输入 `/` 弹出命令菜单）
- 图片支持拖拽上传、粘贴上传

**工具栏按钮**：
```
[B] [I] [U] [S] | [H1][H2][H3] | [• ][1.][☑] | [" ][<>][—] | [🔗][🖼][📊]
粗  斜  下  删    标题级别        列表类型        引用 代码 分割   链接 图片 表格
```

### 2.4 预览区

- Tab 切换不同平台预览
- 预览内容实时更新（debounce 300ms）
- 预览区域模拟对应平台的实际样式（字体、颜色、间距）
- 右上角"在新窗口打开预览"按钮

## 6. 响应式设计

| 屏幕宽度 | 布局 |
|----------|------|
| > 1400px | 三栏（左侧面板 + 编辑区 + 预览区） |
| 1024-1400px | 两栏（左侧面板 + 编辑区，预览通过 Tab 切换） |
| < 1024px | 单栏（移动端不支持发布，仅支持编辑） |

移动端主要用于查看和编辑，发布功能需要桌面 Chrome 浏览器。
